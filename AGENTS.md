# Правила проекта HRNEIROWAY

## Типографика

- Никогда не ставить точки в конце заголовков и подзаголовков
- Правило действует для всех существующих и новых разделов сайта
- Вопросительные и восклицательные знаки сохранять, если они являются частью формулировки

## Язык

- Использовать русские слова, если для иностранного термина есть ясный и привычный русский вариант
- Иностранные названия сохранять только для брендов, платформ, продуктов и терминов без точного русского аналога
